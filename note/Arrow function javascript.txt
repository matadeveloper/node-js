Using normal function:
---------------------------------------------------------------

<script>
function myFunction(){
  let a = 5, b = 4;
  return a+b;
}
console.log(myFunction());
</script>


Using arrow function:
---------------------------------------------------------------
<script>
const myFunction = () =>{
  let a = 5, b = 4;
  return a+b;
}
console.log(myFunction());
</script>
